🎓Application Web de CREATION de CV

📌 Présentation

CREATION CV est une application web développée dans le cadre d’un projet académique en Licence 3 Informatique.
Elle permet aux étudiants de créer, gérer et générer automatiquement leur CV à partir des informations enregistrées dans leur profil.

L’application propose un système complet de gestion d’utilisateurs avec génération dynamique de CV en plusieurs modèles et export au format PDF.
